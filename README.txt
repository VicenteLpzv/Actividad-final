README - sistema de Liga de Fútbol en C#

REQUISITOS DEL SISTEMA

1. Tener instalado .NET 6, 7 o 8 SDK (recomendado).

2. Un editor o IDE compatible con C#:
   - Visual Studio 2022 o superior
   

INSTRUCCIONES PARA EJECUTAR

1. Abrir el proyecto en tu editor/IDE
2. Verificar que todos los archivos .cs estén en la misma carpeta del proyecto:
   - Persona.cs
   - Jugador.cs
   - Equipo.cs
   - Partido.cs
   - Liga.cs
   - Program.cs
3. Asegurate que todos esten en el mismo namespace o sin namespace.

4. Compilar el proyecto:
   - Si usas Visual Studio presiona F5 o selecciona "Iniciar depuración"
  

ESCENARIO DEMOSTRADO EN EL PROGRAMA

- Creación de una liga.
- Creación de dos equipos.
- Creación y asignación de jugadores.
- Creación de un partido.
- Simulación de goles.
- Finalización del partido.
- Impresión de estadísticas y tabla de posiciones


NOTAS IMPORTANTES
- Ha sido necesario crear una clase llamada "Persona" para cumplir con herencia y el polimorfismo
- Si mueves los archivos a carpetas, actualiza los namespaces
